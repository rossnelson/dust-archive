# Welcome to Dust

Dust is a small team of junior developers attempt at a CMS in Rails. 

## Install

`git clone git@github.com:rossnelson/dust.git`

setup the `database.yml` file

`bundle`

`rake db:setup`

`rails s`

## Seeded Admin

navigate to `localhost:3000/dust/login`

- user: `admin`
- pass: `secret`
